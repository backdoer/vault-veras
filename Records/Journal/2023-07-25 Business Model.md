# Business Model


- culture of experimentation
- focus, focus, focus. What’s our key metric?

Questions:
- my fear is that adding more surface area to the app alone will not be the silver bullet. How do we balance succeeding in one of these areas with attempting to go fast and tackle all of the adjacent opportunities? In other words, should we be accelerating and going fast vertically or horizontally?
- Where in the AI value stack should we be competing? Feels like if we can set ourselves up to acquire a lot of proprietary data, we can leverage that to build our own models once it’s cheaper to train an LLM. Moore’s law of cost reduction in computing (2x every 18 months). The cost to train models (GPT spent $400M to train GPT4. The further up the stack we can go, the better opportunity we’ll have. Get in and gain proprietary data so that when models get cheaper, we’ll be well positioned. Social media winner was the third or fourth player, not Friendster and Myspace. 

Data acquisition:
- course content
- examination criteria
- Licensure criteria
- Accreditation criteria (public)
- Wage and employment data (currently collecting)
- school review data
- employer review data

I love it. I think there’s totally a lifecycle there. 

User in high school -> Advertise trade school -> User goes to trade school -> Process tuition -> Supply education? -> Advertise jobs -> Advertise discounted products for field -> Advertise credentialing opportunities -> Administer examinations? -> User graduates ->  Ask for feedback on school -> User gets job -> Track placement -> Ask for feedback on employer -> Repeat
