# Levels of Engagement

- User has "claimed profile" (updated something)
  - They could've done this over text
  - Does filling out an outcome form count as claiming profile? I think so.
  - This could be skills, creds, exp. resume, preferences, etc.
- Profile has been enriched
  - This could've been done by the school, or kenn garff, or some other third party but still useful
- User has logged in

Other ideas:
- User has been through onboarding
- User has personally added skills
- User has personally added credentials
- User has personally added preferences
- User filled out outcome form
- User has filled out resume

* This could just be an event table that stores everything a user has done?

All of these things could've been added by a third party or the user and it's likely worthwhile to keep track. Do we keep track of every resource type (skill, cred, pref, etc.) and where they came from? Or is enriched vs. claimed sufficient?
