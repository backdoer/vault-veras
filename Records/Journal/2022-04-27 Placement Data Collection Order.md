# Placement Data Collection Order

Strategies:
1. Placement Hub (manual) ---> Gradually Add Automations
2. Labor Marketplace (1 automated input into placement) ---> Gradually Become Placement Hub

First Approach:
1. Get students into system
- Read/Write SIS integrations (one at a time)
- Fields: name, program, phone number, email, birthday, start date, required hours/courses/credits, completed hours/courses/credits, end date, end code (graduated, dropped out, etc.), instructor(s), current placement status
2. Create tracking hub
- Assignments
- Actionability (recent graduations, upcoming graduations, etc.)
- Timeline
- Imports of third party employment reports (USHE/DWS)
- Regular emails to assigned admins/instructors around updates (placed and in need of placement)
- Leaderboard on tracking?
- Reporting
3. Labor Marketplace
- Job boards
   - Embedded (RSS)
   - Centralized
- Post/Import Job(s)
- ATS (w/ incentives to report back)
4. Zoe, the career mentor bot (student reports & JobWise signing bonus)

7. Current Payroll (HR) integrations (one at a time)
